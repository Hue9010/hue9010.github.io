원격 저장소
=========

**원격 저장소에 브랜치 생성**

> git push origin 브랜치 이름

**원격 저장소에 브랜치 삭제**

> git push origin :브랜치 이름  

추가로 로컬에서 삭제  
------------------

**삭제 할 브랜치 외의 브랜치로 이동**  

> git checkout 브랜치 이름  

**로컬 브랜치 삭제**  

> git branch -D 브랜치 이름
